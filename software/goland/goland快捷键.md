
## 快捷键
### Mac 快捷键
![](images/Mac按键.png)
* 删除 delete
* 右删除 fn delete
* 右制表符 delete
* 左制表符 fn delete
* Home fn ←
* End fn →
* Page Up fn ↑
* Page Down fn ↓
* 切换桌面 control ↑↓←→

### Shortcuts 捷径
* 重命名 shift F6
* 收缩展开代码 command -/+

### General 常用
* 折叠左边目录树 command 1 
* 定位文件所在目录树 alt F1
* 设置 command ,

### Usage Search 变量搜索
* 哪些地方使用到 command F7

### Editing 文本编辑
* 删除一行 control y/command delete(mac)
* 查看文档 control j
* 查看参数 control p
* 大小写 command shift u
* 查看最近复制 command shift v
