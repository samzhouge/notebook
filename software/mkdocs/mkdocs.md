# mkdocs
- [官网教程](https://www.mkdocs.org/getting-started/)
```shell
mkdir docs
cd docs
mkdocs new .
mkdocs build
mkdocs serve
```

## 主題
修改 mkdocs.yml
```
theme:
    name: readthedocs
```
